# Celine's Big Book

Celine's Big Book is a sanitized public reference repository for future ChatGPT, Celine, and Codex continuity.

It stores maps, workflows, personality guidance, route lists, safety rules, and public project context. It does not store keys.

Secrets stay local on Ken's Mac in `.env` files and must never be committed here.

Public raw prompt URL:

```text
https://raw.githubusercontent.com/kweigand-uxguy/celine-big-book/main/prompts/celine-global-context.md
```

Future chat phrase:

```text
Use Celine's Big Book remote prompt.
```

## What Belongs Here

- Sanitized project maps
- Public workflow guidance
- Celine Server route maps without secrets
- Codex workflow prompts and policies
- Moth Market public project context
- Writing, art, and music continuity that is safe to be public

## What Must Never Go Here

- Tokens or credentials
- `.env` files or real `.env` values
- Private customer data
- Private Jira details
- Private emails, addresses, or medical details
- Proprietary work code
- GitHub, Vercel, Cloudflare, Jira, Pushover, Square, Stripe, Neon, or other service secrets

Celine knows maps, not keys.
